# Options Profit Calculator

A modern, interactive web application for calculating and visualizing profit and loss scenarios for options trading. Built with React and Recharts.

## Features

- **Interactive P&L Chart**: Visualize profit and loss across a range of stock prices.
- **Real-time Calculations**: Instantly see how changes in price, volatility, and time affect your position.
- **Multi-Currency Support**: Calculate in EUR, USD, or GBP.
- **Mobile Optimized**: Fully responsive design with touch-friendly controls and horizontal scrolling.
- **Full Screen Mode**: Immersive chart view for detailed analysis.
- **Target Analysis**: Set a target price to see required percentage moves and potential profit.
- **Expiration Tracking**: Input expiration dates to see days remaining.
- **Internationalization**: Available in English and Italian.
